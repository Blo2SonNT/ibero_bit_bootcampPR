GIT

Recuerden que para esto se necesita el software de git

Comando Uso
git init Inicializar el seguimiento de git sobre el proyecto
git add . || git add index.html Indica a git que el o los archivos que seran subidos al repositorio
git commit -m "comentario" Realiza la confirmacion para subir los archivos previamente agregados con git add
git remote Enlaza nuestro proyecto con el repositorio creado en GitHub
git push Da la instruccion para proceder a subir los archivos del proyecto al repositorio
git branch Creacion de rama
git checkout Permite seleccionar la ramma en la que se desea trabajar
git clone Descarga un repositorio ya existente
git merge Permite unir los cambios de dos ramas
